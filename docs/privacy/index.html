베베컵 개인정보처리방침

최종 업데이트: 2026-05-24

1. 서비스

베베컵은 부모 또는 보호자가 직접 선택한 로컬 아기 사진으로 이상형 월드컵을 만들고 인화 후보를 정리하는 Android 앱입니다.

2. 처리하는 정보

앱은 사용자가 Android Photo Picker에서 직접 선택한 사진의 로컬 URI와 앱 사용 중 생성한 후보/월드컵 기록을 기기 내부 저장소에 저장합니다.

3. 수집 및 공유

베베컵은 사진 파일, 사진 URI, 사용 기록, 기기 식별자, 위치 정보, 연락처, 계정 정보를 외부 서버로 전송하거나 제3자와 공유하지 않습니다. 분석, 광고, 추적 SDK를 사용하지 않습니다.

4. 외부 인화 사이트

앱은 사용자가 명시적으로 선택한 경우 외부 인화 사이트를 브라우저로 열 수 있습니다. 이 경우 이후의 개인정보 처리와 주문 절차는 해당 외부 사이트의 정책을 따릅니다. 베베컵은 사진 파일을 외부 인화 사이트에 자동 업로드하지 않습니다.

5. 보관 및 삭제

사진 후보와 기록은 사용자의 기기에만 저장됩니다. 사용자는 앱 안에서 후보를 삭제하거나 Android 설정에서 앱 데이터를 삭제할 수 있으며, 앱을 삭제하면 로컬 데이터도 함께 삭제됩니다.

6. 보안

앱 데이터는 Android 앱 샌드박스 안에 저장됩니다. 외부 서버 전송이 없으므로 전송 구간 암호화 대상 데이터는 없습니다.

7. 어린이 데이터

베베컵은 어린이를 대상으로 제작된 앱이 아니며, 18세 이상의 부모 및 보호자가 사용하도록 설계되었습니다. 앱은 어린이로부터 어떠한 개인정보도 직접 수집하지 않습니다.

8. 정책 변경

본 방침이 변경되는 경우 본 페이지를 통해 공지합니다. 중요한 변경은 최종 업데이트 일자로 표시됩니다.

Bebecup Privacy Policy

Last updated: 2026-05-24

1. Service

Bebecup is an Android app that lets parents and guardians build a photo tournament from local baby photos they personally choose, and organize favorite photos for printing.

2. Information processed

The app stores, on the device only, the local URIs of photos the user explicitly selects through the Android Photo Picker, together with the tournament and candidate records the user creates while using the app.

3. Collection and sharing

Bebecup does not transmit photo files, photo URIs, usage records, device identifiers, location, contacts, or account information to any external server, and does not share them with third parties. No analytics, advertising, or tracking SDKs are used.

4. External print sites

The app may open external print sites in the user's browser when the user explicitly chooses to do so. Any subsequent data processing and ordering on those sites is governed by the policies of those external services. Bebecup does not automatically upload photo files to external print sites.

5. Retention and deletion

Photo candidates and records are stored only on the user's device. The user can delete candidates from inside the app, clear app data from Android system settings, or uninstall the app, which also removes the local data.

6. Security

App data is stored inside the Android app sandbox. Because no data is transmitted to external servers, no in-transit data is subject to encryption requirements.

7. Children's data

Bebecup is not designed for children. It is intended for use by parents and guardians aged 18 or older. The app does not directly collect any personal information from children.

8. Changes to this policy

Changes to this policy will be communicated through this page. Material changes will be reflected in the "Last updated" date above.
