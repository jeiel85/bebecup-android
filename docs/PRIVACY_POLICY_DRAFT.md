# 베베컵 개인정보처리방침 초안

최종 업데이트: 2026-05-24

이 문서는 Play Console 제출 전 공개 웹 페이지로 게시하기 위한 초안입니다. 실제 게시 페이지는 [docs/privacy/index.html](privacy/index.html)이며, 공개 URL은 https://jeiel85.github.io/bebe-cup/privacy/ 입니다. 이 마크다운은 사람 읽기용 원본으로 보관합니다.

## 1. 서비스

베베컵은 부모 또는 보호자가 직접 선택한 로컬 아기 사진으로 이상형 월드컵을 만들고 인화 후보를 정리하는 Android 앱입니다.

## 2. 처리하는 정보

앱은 사용자가 Android Photo Picker에서 직접 선택한 사진의 로컬 URI와 앱 사용 중 생성한 후보/월드컵 기록을 기기 내부 저장소에 저장합니다.

## 3. 수집 및 공유

베베컵은 사진 파일, 사진 URI, 사용 기록, 기기 식별자, 위치 정보, 연락처, 계정 정보를 외부 서버로 전송하거나 제3자와 공유하지 않습니다. 분석, 광고, 추적 SDK를 사용하지 않습니다.

## 4. 외부 인화 사이트

앱은 사용자가 명시적으로 선택한 경우 외부 인화 사이트를 브라우저로 열 수 있습니다. 이 경우 이후의 개인정보 처리와 주문 절차는 해당 외부 사이트의 정책을 따릅니다. 베베컵은 사진 파일을 외부 인화 사이트에 자동 업로드하지 않습니다.

## 5. 보관 및 삭제

사진 후보와 기록은 사용자의 기기에만 저장됩니다. 사용자는 앱 안에서 후보를 삭제하거나 Android 설정에서 앱 데이터를 삭제할 수 있으며, 앱을 삭제하면 로컬 데이터도 함께 삭제됩니다.

## 6. 보안

앱 데이터는 Android 앱 샌드박스 안에 저장됩니다. 외부 서버 전송이 없으므로 전송 구간 암호화 대상 데이터는 없습니다.
