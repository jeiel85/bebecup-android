베베컵

직접 고른 아기 사진으로 만드는 가족 사진 월드컵

A family photo tournament made from baby photos you choose.

주요 기능 Key features

📷

사진 후보 관리

Android Photo Picker로 가족이 직접 고른 아기 사진만 후보로 등록합니다. 목업·샘플 사진은 넣지 않습니다.

🏆

이상형 월드컵

토너먼트로 좋아하는 사진을 골라가며 최고의 한 장을 뽑습니다. 우승 사진은 명예의 전당에 기록됩니다.

🖼️

인화 후보 정리

가족이 함께 인화할 후보를 따로 모아두고, 결정한 사진만 외부 인화 사이트로 가져갈 수 있습니다.

🔒

기기 안에서만

선택한 사진의 로컬 URI와 월드컵 기록은 기기 내부 저장소에만 보관됩니다. 외부 서버 업로드 없음.

개인정보 보호 Privacy first

베베컵은 분석·광고·추적 SDK를 사용하지 않습니다. 사용자가 명시적으로 외부 인화 사이트를 열기 전까지 사진 파일은 기기 밖으로 나가지 않습니다.

아이의 사진은 가족의 것입니다. 베베컵은 가족 안에서 사진을 정리할 수 있게 돕는 도구일 뿐, 광고 노출이나 데이터 수익화의 대상이 아닙니다.

자세한 내용은 개인정보처리방침에서 확인할 수 있습니다.
