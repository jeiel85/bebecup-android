베베컵은 부모와 보호자가 직접 고른 실제 아기 사진으로 이상형 월드컵을 만들고, 가족이 함께 인화 후보를 정리할 수 있는 Android 앱입니다.

사진은 Android Photo Picker를 통해 사용자가 직접 선택합니다. 앱은 첫 실행 시 목업 사진이나 샘플 사진을 넣지 않으며, 선택한 로컬 이미지 URI와 월드컵 기록만 기기 내부에 저장합니다.

주요 기능
- 직접 선택한 사진 후보 관리
- 사진 이상형 월드컵 진행
- 우승 사진과 인화 후보 정리
- 외부 인화 사이트 브라우저 열기

베베컵은 사진 파일을 임의로 외부 서비스에 업로드하지 않습니다. 외부 인화 사이트 연결은 사용자가 명시적으로 브라우저를 여는 방식으로만 제공됩니다.

분석, 광고, 추적 SDK 없이 가족 사진을 기기 안에서 조용히 정리할 수 있도록 설계했습니다.
